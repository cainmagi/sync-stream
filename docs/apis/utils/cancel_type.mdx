```python
_data: Any = cancel_type(data: Any)
```

Remove the typehint of a value with a known type.

It is equivalent with

```python
_data = cast(Any, data)
```

## Arguments

### Requires

| Argument | Type  |             Required              | <center>Description</center>   |
| :------: | :---: | :-------------------------------: | :----------------------------- |
|  `data`  | `Any` | <InlineIcon icon={vsiCheckIcon}/> | A value that has any notation. |

### Returns

| Argument | Type  | <center>Description</center>                                   |
| :------: | :---: | :------------------------------------------------------------- |
| `_data`  | `Any` | The same input `data`, while the notation is replaced by `Any`. |
