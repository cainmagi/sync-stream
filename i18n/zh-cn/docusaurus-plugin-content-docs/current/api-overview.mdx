以下模块是可选的。要安装它们，请在安装时添加对应参数。不同的参数可以一起使用。

- `file`: 需要`pip install syncstream[file]`
- `host`: 需要`pip install syncstream[host]`

## 模块 {#modules}

|                                           成员                                           | <center>说明</center>                                                                                              |
| :--------------------------------------------------------------------------------------: | :----------------------------------------------------------------------------------------------------------------- |
|     [<IconObjType type="module" hasText={true} text="base"/>](./apis/base/index.mdx)     | 用于不同模块的基本工具。                                                                                           |
|    [<IconObjType type="module" hasText={true} text="mproc"/>](./apis/mproc/index.mdx)    | 透过Python标准库实现的、线程和进程安全的信息同步。                                                                 |
|     [<IconObjType type="module" hasText={true} text="file"/>](./apis/file/index.mdx)     | 文件信息同步，基于[`fasteners`<IconExternalLink/>][link-fasteners]实现。                                           |
|     [<IconObjType type="module" hasText={true} text="host"/>](./apis/host/index.mdx)     | 网络服务信息同步，基于[`Flask`<IconExternalLink/>][link-flask]和[`urllib3`<IconExternalLink/>][link-urllib3]实现。 |
| [<IconObjType type="module" hasText={true} text="webtools"/>](./apis/webtools/index.mdx) | 用来实现网络服务的共享工具。                                                                                       |
|    [<IconObjType type="module" hasText={true} text="utils"/>](./apis/utils/index.mdx)    | Python标准库的一些扩展。                                                                                           |
