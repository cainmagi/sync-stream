A group of messages.

Used for wrapping muliple `str` lines (info), a warning object, or an exception object.

This class would be used by [`mproc.LineProcMirror`](../mproc/LineProcMirror.mdx) and [`host.LineHostMirror`](../host/LineHostMirror.mdx) automatically. In most cases, users do not need to use this class directly.

## Arguments

### Requires

| Argument |                     Type                     | <center>Description</center>                                                                                                              |
| :------: | :------------------------------------------: | :---------------------------------------------------------------------------------------------------------------------------------------- |
|  `data`  | `Sequence[str]`, or `Warning` or `Exception` | The grouped messages provider. The input value would be parsed as formatted lines. The parsed messages are compatible with serialization. |

## Methods

### <IconObjType type="method"/> `serialize`

```python
jdata: dict[str, Any] = gpm.serialize()
```

Serialize this message item into a JSON compatible dict.

#### Returns

| Argument |  Type  | <center>Description</center>                                                                                                                |
| :------: | :----: | :------------------------------------------------------------------------------------------------------------------------------------------ |
| `jdata`  | `dict` | The serialized dictionary. This dictionary could be converted back to `GroupedMessage` object by the [`deserialize`](#-deserialize) method. |

### <IconObjType type="method"/> `deserialize`

```python
gpm: GroupedMessage = GroupedMessage.deserialize(jdata: dict[str, Any])
```

#### Requires

| Argument |  Type  | <center>Description</center>                                                                                 |
| :------: | :----: | :----------------------------------------------------------------------------------------------------------- |
| `jdata`  | `dict` | The serialized dictionary. This dictionary is JSON compatible, and could be transmitted by the web packages. |

#### Returns

| Argument |       Type       | <center>Description</center>                                                                |
| :------: | :--------------: | :------------------------------------------------------------------------------------------ |
|  `gpm`   | `GroupedMessage` | The deserialized object. This object could be converted to formatted strings by `str(gpm)`. |
