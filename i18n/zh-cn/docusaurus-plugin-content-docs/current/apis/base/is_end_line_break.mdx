```python
flag: bool = is_end_line_break(val_str: str)
```

用来检查`str`是否由断行符收尾的函数。

该检查透过以下工具实现

<IconLink href="https://docs.python.org/zh-cn/3/library/stdtypes.html#str.splitlines" />

该函数用于修复直接调用`str.splitlines`后被移去的结尾断行符信息。

## 参数 {#arguments}

### 输入 {#requires}

|   参数    | 类型  |               必选                | <center>说明</center> |
| :-------: | :---: | :-------------------------------: | :-------------------- |
| `val_str` | `str` | <InlineIcon icon={vsiCheckIcon}/> | 要验证的字符串。      |

### 输出 {#returns}

|  参数  |  类型  | <center>说明</center>                    |
| :----: | :----: | :--------------------------------------- |
| `flag` | `bool` | 若为`True`，则所验证的字符以断行符收尾。 |
