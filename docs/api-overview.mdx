The follolowing modules are optional. To install them, please add the arguments during the installation. Different arguments could be used in the same time.

- `file`: need to use `pip install syncstream[file]`
- `host`: need to use `pip install syncstream[host]`

## Modules

|                                          Member                                          | <center>Description</center>                                                                                                                                                   |
| :--------------------------------------------------------------------------------------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|     [<IconObjType type="module" hasText={true} text="base"/>](./apis/base/index.mdx)     | The shared basic tools for different modules.                                                                                                                                  |
|    [<IconObjType type="module" hasText={true} text="mproc"/>](./apis/mproc/index.mdx)    | The thread-safe and process-safe message synchronization, implemented based on the python stdlib.                                                                              |
|     [<IconObjType type="module" hasText={true} text="file"/>](./apis/file/index.mdx)     | The file-lock-based message synchronization, implemented based on [`fasteners`<IconExternalLink/>][link-fasteners].                                                            |
|     [<IconObjType type="module" hasText={true} text="host"/>](./apis/host/index.mdx)     | The web-service-based message synchronization, implemented based on [`Flask-RESTful`<IconExternalLink/>][link-flask-restful] and [`urllib3`<IconExternalLink/>][link-urllib3]. |
| [<IconObjType type="module" hasText={true} text="webtools"/>](./apis/webtools/index.mdx) | The shared utilities for web services.                                                                                                                                         |
|    [<IconObjType type="module" hasText={true} text="utils"/>](./apis/utils/index.mdx)    | Extensions of the Python STL.                                                                                                                                                  |
